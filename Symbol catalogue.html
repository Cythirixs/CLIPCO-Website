All of the icons, in one place

The CLIPCO Icon Catalogue

To use these Icons on other pages of the site, include the name and category of the icon and "icon" in the class of a span tag, and the css should handle the rest. Icons are treated as text so any text color, size, etc. adjustments will also affect the icons.

Example:

HTML: 
<span class="icon education-portal cs-icon-education-portal-3"></span>
result:

Example:

HTML: 
<span class="icon education-portal cs-icon-education-portal-3 red"></span>
CSS:
.red {
    color: red;
}
result:
