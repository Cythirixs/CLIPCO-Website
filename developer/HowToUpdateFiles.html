How to update pages

Reminder: make sure changes have been approved through the Github process, and merged into master before pushing to the site.
There are two ways to update pages on the site.

  • Through Cpanel

    1. Log in through cpanel at cusdclipco.org/cpanel
    2. From the dashboard, select File Manager
    3. The file manager should open up to public_html. Navigate to the directory with the file you wish to update. then Select Upload.
    4. Drag in or select your file into the upload dialogue window.

      If it asks you if you want to overwite existing files, just click Yes.

    And that's it!


  • Using Cyberduck

    If you choose to use Cyberduck to update the website, then we first need to set up Cyberduck to connect to the website. You can find instructions on how to do that through Cpanel here.

    Once you have cyberduck installed and set up then you navigate to the appropriate directory (usually public_html) and drag your files in from your computer.

If you followed all of the above steps, then your file should be uploaded! navigate to the file you changed on the site to check if it's updated. If your changes don't appear immediately, refresh the page a couple of times. If that still doesn't work then you may need to empty your cache.
